# 一、完整版全局游戏规则
## 1. 基础核心设定
1. 双方队伍
   - `ctf_defend` 防守方（绿色）：开局持有正版夺旗旗帜
   - `ctf_attack` 进攻方（红色）：目标夺取/销毁正版旗帜
2. 正版旗帜唯一标识
   - 每局开局随机生成数字唯一ID，NBT：`custom_data:{ctf_flag:1b,flag_unique_uuid:本局ID}`
   - 全局判定、追踪、心跳仅识别带正确UID的正版旗
   - 无匹配UID的钻石为**诱饵假旗**：无追踪、无发光、拾取/销毁不触发胜负，防守合法迷惑战术
3. 原版指南针限制：追踪marker只指向**优先级最高、距离最近**的正版旗；优先级：持旗玩家 > 地面掉落旗 > 展示框旗帜

## 2. 三大胜负判定（任意一条触发立刻结束，优先级相同）
### 进攻方胜利
1. 任意进攻玩家背包持有**正版旗帜**
2. 地图内不存在任何正版旗帜（被岩浆/虚空/爆炸/仙人掌销毁）
3. 正版旗帜存入任意玩家私人末影箱（竞技末影规则开启时）
4. 旗帜心跳机制开启：正版旗帜脱离玩家背包累计满60秒（1200刻）
5. 检测到任何正版旗帜（复制多份不惩罚，但任意一面触发上面任意条件直接输）

### 防守方胜利
全局倒计时耗尽，全程未触发任意进攻胜利条件

### 结算统一效果
1. 全屏大Title标题+配色区分胜负
2. 全服播放末影龙死亡音效
3. 自动清空所有正版/假旗帜、追踪指南针、持旗头顶文字、发光效果

## 3. 四大可选开关机制（管理员自由开启/关闭，独立互不干扰）
### 机制1：烫手山芋（持旗高亮追踪）
开关：`hotpotato_switch`
- 开启：持有正版旗玩家持续隐藏粒子发光、头顶金色「🏴持旗者🏴」悬浮文字；进攻方获得专用追踪指南针指向最近正版旗
- 关闭：无发光、无头顶文字、自动回收追踪指南针

### 机制2：全域载体追踪（箱子/掉落/展示框均可定位）
开关：`flag_track_all_state`
- 开启：无论正版旗在玩家背包、地面、展示框，指南针均可锁定坐标
- 关闭：仅玩家手持正版旗时指南针生效，落地/入容器则追踪失效

### 机制3：旗帜心跳（防黑曜石永久龟缩）
开关：`flag_heartbeat_switch`
- 开启：正版旗不在玩家背包时持续倒计时，满60秒直接进攻胜利；底部黄色倒计时警告，最后10秒橙色加粗警告
- 关闭：旗帜可永久存放容器固守，无离线计时

### 机制4：末影箱规则（双模式切换）
开关：`enderchest_flag_rule`
1. 竞技模式（默认）：正版旗存入私人末影箱直接判进攻胜利（末影箱私有无法被进攻获取）
2. 休闲共享模式：数据包同步全服末影箱物品，旗帜可正常存取销毁，无存旗惩罚

### 机制5：神仙打架进阶模式（总开关 `godmode_switch`）
开启后：
1. 自动关闭烫手山芋、全域追踪、旗帜心跳所有自动辅助功能
2. 清除全部发光、头顶文字、追踪指南针、marker标记
3. 全体玩家给予OP指令权限，自由使用定位、tp、summon、fill等指令博弈
4. **底层胜负、正版UID校验规则完全锁死不失效**
5. 玩家只能手动写指令检索正版旗帜，无系统自动导航

## 4. 复制旗帜规则（核心平衡设计）
1. 防守方复制完整带UID正版旗：不触发系统惩罚，但属于纯负面自损战术
   - 多目标同时发光、多坐标被指南针锁定
   - 任意一面被夺取/销毁/离线超时直接输掉对局
2. 无UID假旗：合法诱饵，不参与任何追踪、胜负判定，仅用于视觉迷惑进攻方

## 5. 视觉美化全配套
1. 比赛开局：全屏金色Title大标题+规则副标题
2. 胜负结算：对应红/绿色胜负大标题
3. 切换烫手山芋开关：底部ActionBar小字即时提示状态
4. 全局倒计时剩余≤30秒：红色闪烁ActionBar紧急警告
5. 烫手开启时，持旗玩家头顶永久悬浮金色标识
6. 心跳开启时底部持续显示离线倒计时，临近超时橙色警告

## 6. 游玩四种标准组合模式
1. 休闲建造局：心跳关闭+全域追踪开启+烫手关闭+末影休闲模式
   可黑曜石密室长期藏旗，主打阵地建造防守
2. 平衡竞技局（推荐）：心跳开启+全域追踪开启+烫手关闭+末影竞技模式
   禁止无限龟缩，必须轮换持旗拉扯，阵地攻防平衡
3. 高速追逐局：心跳开启+烫手开启+全域追踪关闭+末影竞技模式
   旗帜落地即丢失追踪，只能追击活人持旗者
4. 神仙指令大战：开启神仙模式，所有自动辅助失效，全员OP手动指令搜旗博弈

# 二、全套管理员OP指令清单
## 初始化（新存档仅执行1次）
`/function ctf:init`

## 对局控制
开启一局：`/function ctf:start`
强制重置作废对局：`/function ctf:reset`

## 队伍加入
防守方：`/team join ctf_defend 玩家名`
进攻方：`/team join ctf_attack 玩家名`

## 可选机制开关
烫手开启：`/function ctf:hotpotato_on`
烫手关闭：`/function ctf:hotpotato_off`

全域追踪开启：`/function ctf:track_all_on`
全域追踪关闭：`/function ctf:track_all_off`

心跳防龟缩开启：`/function ctf:heartbeat_on`
心跳防龟缩关闭：`/function ctf:heartbeat_off`

末影箱竞技模式：`/function ctf:enderchest_compete_on`
末影箱休闲共享模式：`/function ctf:enderchest_fun_on`

神仙指令大战开启：`/function ctf:godmode_on`
神仙模式关闭恢复常规：`/function ctf:godmode_off`
